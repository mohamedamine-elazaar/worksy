# Work Back

Backend server scaffold with Express and MongoDB.

## Setup

1. Install dependencies:

```
npm install
```

2. Ensure MongoDB is running locally or update `MONGO_URI` in `work back/.env`.

## Development

Run in development with auto-restart:

```
npm run dev
```

## Production

Run directly:

```
npm start
```

## Endpoints

- `GET /api/health` — health check
- `GET /api/example` — sample endpoint
